Handles the processing of an event sample. An Analyzer is built for each Config.Analyzer present in sequence. The Looper can then be used to process an event, or a collection of events.

Parameters: name : name of the Looper, will be used as the output directory name config : process configuration information, see Config nEvents : number of events to process. Defaults to all. firstEvent : first event to process. Defaults to the first one. nPrint : number of events to print at the beginning

def loop(self):

Loop on a given number of events.

At the beginning of the loop, Analyzer.beginLoop is called for each Analyzer. At each event, self.process is called. At the end of the loop, Analyzer.endLoop is called.
