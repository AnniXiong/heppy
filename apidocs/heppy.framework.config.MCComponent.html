A component for Monte Carlo

In addition to the attributes present in the base Component class, MCComponent contains:

* xSection : the cross-section for the corresponding process * nGenEvents : the number of generated events * effCorrFactor : an efficiency correction factor * addWeigth : an additional weight * intLumi : target integrated luminosity
