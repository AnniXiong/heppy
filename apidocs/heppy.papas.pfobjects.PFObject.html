heppy.papas.pfobjects.PFObject(object) class documentation

Part of heppy.papas.pfobjects (View In Hierarchy)

Known subclasses: heppy.papas.pfobjects.Cluster, heppy.papas.pfobjects.Track

Base class for all particle flow objects (tracks, clusters, etc). Particle flow objects of different types can be linked together forming graphs called "blocks".

attributes: linked : list of PFObjects linked to this one locked : already used in the particle flow algorithm block_label : label of the block the PFObject belongs to. The block label is a unique identifier for the block.
