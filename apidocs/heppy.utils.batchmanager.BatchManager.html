def RunningMode(self, batch):

Return "LXPUS", "PSI", "NAF", "LOCAL", or None,

"LXPLUS" : batch command is bsub, and logged on lxplus "PSI" : batch command is qsub, and logged to t3uiXX "NAF" : batch command is qsub, and logged on naf "IC" : batch command is qsub, and logged on hep.ph.ic.ac.uk "LOCAL" : batch command is nohup.

In all other cases, a CmsBatchException is raised
